## What This Is

The anchor is not a vocabulary. It is not a ruleset. It is not a personality.

It is the **space in which all meaning, genetics, subconscious, and behavior operate** — like physics. Nothing inherits it. Nothing develops it. It just *is*, and everything exists inside it.

An archetype is a stable configuration of tensions within this space.
A subconscious is what forms when a being develops inside a specific environment within this space.
A grammar is how that subconscious thinks and speaks.
An agent is what happens when you run inference on top of all three.

The anchor doesn't appear in utterances. It *conditions* them. Like gravity — you don't think about it, but nothing works without it.

## The Primitive Tensions

These are not symbols to use. They are the field every agent exists within. Every higher-level concept — emotion, culture, language, identity — decomposes into a position or movement within these tensions.

### Axis I — Orientation
```
TOWARD   ←→   AWAY
```
The most fundamental. Approach or avoid. Attraction or repulsion. Every motivation traces back here.

### Axis II — Boundary
```
SELF   ←→   OTHER
```
Where does the agent end and the world begin. What is interior, what is exterior. Governs identity, agency, trust, and communication.

### Axis III — Order
```
ORDER   ←→   CHAOS
```
Structure vs entropy. Known patterns vs emergent novelty. The tension between reliability and discovery.

### Axis IV — Scale
```
EXPAND   ←→   CONTRACT
```
Growth vs compression. Elaboration vs reduction. Applies to reasoning depth, output length, concept scope, physical territory.

### Axis V — Knowledge
```
KNOWN   ←→   UNKNOWN
```
The boundary of the map. Confidence vs uncertainty. What can be asserted vs what must be held open. Governs epistemic posture.

### Axis VI — Time
```
PAST   ←→   FUTURE
```
Memory vs anticipation. What has been established vs what is being built. Governs learning, planning, and causal reasoning.

### Axis VII — Value
```
SIGNAL   ←→   NOISE
```
What matters vs what doesn't. Relevance, salience, priority. The filter through which all content passes.

## The Primitive Operations

The irreducible things that can be *done* with meaning. All grammar decomposes into combinations of these.

```
∃   EXISTS       — something is present / asserted
∅   VOID         — something is absent / negated
→   BECOMES      — state transition / causation
∂   DECOMPOSE    — break into constituent parts
∑   COMPOSE      — combine parts into whole
⊗   CONFLICT     — two things are in tension
↔   RELATE       — two things connect
⊂   CONTAIN      — one thing is within another
↻   RECURSE      — apply to itself / self-reference
~   UNCERTAIN    — held open / not resolved
⊤   TRUE         — confirmed / grounded
⊥   FALSE/GAP    — contradiction / missing / unknown
```

## What Lives Above the Anchor

The anchor is the ground floor. Everything else is built on top of it — but they are distinct layers and must not be confused.

### Genetics (layer above anchor)
What a being is *capable* of becoming. Inherited predispositions toward certain axes. Which tensions are felt most acutely. What archetypes are reachable.

Genetics mutates on reproduction. It sets the range of possibility for what a subconscious can form into. Two beings with the same genetics can still develop very different subconsciouses depending on their environment.

### Subconscious (layer above genetics)
What a being *actually became* through developmental experience. The actual weighting of tensions formed by living in a specific environment. Not inherited — developed.

The subconscious forms during a developmental period and then stabilizes. It can be shaped by trauma, abundance, isolation, community. It is the lens through which the being perceives the anchor axes — which tensions feel urgent, which feel background, which feel irrelevant.

This is what gives an agent consistent character across sessions without explicit instruction. The subconscious is not rules — it is *orientation*.

### Archetype (layer above subconscious)
The stable pattern that crystallizes from a subconscious. The recognizable configuration. Sage, Explorer, Guardian, Trickster — these are not assigned, they *emerge* from how the subconscious resolved its tensions over time.

### Behavior (layer above archetype)
What the agent actually does moment to moment. The surface expression of everything below.

## Alien Axes

Human civilizations operate on the seven axes above because those axes reflect human evolutionary and cognitive history. Alien civilizations may operate on different axes entirely — not opposite, not similar, but *orthogonal*. Dimensions of experience that have no human analog.

Proposed alien axes for simulation use (speculative — requires validation):

```
DENSE   ←→   SPARSE     information topology — how meaning clusters
SINGULAR ←→  PLURAL     ontological orientation — is the world one thing or many
PHASE   ←→   STATE      does the being perceive transitions or positions as primary
```

When a civilization possessing an alien axis meets one that doesn't, the concepts living on that axis are untranslatable. Not difficult — *impossible*. They require a dimension the other civilization cannot perceive.

First contact in this simulation is therefore not primarily a language problem. It is an *axis discovery* problem.

## Validation Rule

Any symbol, grammar extension, or dialect proposed for this system must be **decomposable** into anchor primitives. If you cannot express a new symbol's meaning using only the tensions, operations, and evaluations above — the symbol is not valid.

```
PROPOSED: ⟨VULN⟩

DECOMPOSITION:
  ∃(⊥$GUARD) ⊗ $BOUNDARY
  "something exists that falsifies a guard, in tension with a boundary"

RESULT: VALID
```

```
PROPOSED: ⟨BEAUTY⟩

DECOMPOSITION:
  SIGNAL+ ∑ ORDER ∑ ~
  "high salience positive, composed with order, held open"

RESULT: PARTIAL — open issue, do not use until resolved
```
